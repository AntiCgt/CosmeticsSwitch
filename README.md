CosmeticsSwitch for Gorilla Tag
Mod that lets you hide/show other players' cosmetics with a laser gun. Works only on your screen - others don't see changes.

What it does
Hide cosmetics that block your view

Toggle visibility per player

Works with all cosmetic types

How to use
Hold right trigger to activate laser

Aim at player

Their cosmetics toggle on/off

First shot hides, second shows. States save per player.
